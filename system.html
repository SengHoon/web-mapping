시스템 요구 사항

최소 사양
OS: 64-bit Windows 7
CPU: Intel Core i3-6300 / AMD FX-4350
RAM: 6GB
GPU: NVIDIA GeForce GT 640 / Radeon HD 7700
GPU RAM: 1GB

권장 사양
OS: 64-bit Windows 10
CPU: Intel i5 3570K or equivalent
RAM: 8GB
GPU: Nvidia GeForce GTX 970 / AMD Radeon R9 290
GPU RAM: 8GB

상기한 바와 같이 그래픽 성능을 상당히 요구하는 게임이다. 최소 사양으로 플레이 시 그래픽 칩셋의 한계 때문에 최저옵션에 해상도도 최대한 낮게 설정해도 30프레임 방어가 불가능하다. 하지만 GTX750TI에서는 어느정도 옵션타협을 본다면 1080P에서도 원활하게 굴러간다. 참고 생각 외로 CPU 리소스를 그렇게 크게 요구하지 않는다. 상기된 최저 사양의 CPU급 이상만 되면 그래픽 칩셋 사양만 충족할 시 상대적으로 안정적인 플레이가 가능할 것으로 보인다.[5] AMD Phenom CPU를 비롯한 SSE 4.1 미지원 구형 CPU에서는 그래픽카드와 관계없이 현재 실행이 불가능하다.
